# fab-scripts

## Descripción
Este proyecto consiste en un repositorio de ejemplo para aprender a usar Git y GitHub.  
Incluye un script en Python llamado mi_script.py que realiza operaciones de multiplicación.  
El objetivo es mostrar cómo un programa sencillo, siguiendo las reglas de Commit.


## Instalación
Clona el repositorio:
```bash
git clone https://github.com/Ashley213TT/fab-scripts.git

Entra en la carpeta del repositorio:
cd fab-scripts

- Verifica que tienes Python instalado:
python --version


- Ejecuta el script:
python mi_script.py



## Ejemplo de uso
El script primero muestra el resultado de 2 x 2 y luego permite al usuario ingresar dos números para multiplicarlos.
Ejemplo de ejecución
$ python mi_script.py
El resultado de 2 x 2 es: 4
Ingresa un número: 5
Ingresa otro número: 3
El resultado es: 15
